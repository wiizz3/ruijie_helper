# 校园网认证助手

针对锐捷认证客户端对有线用户禁止使用热点和虚拟机问题而开发网络管理小工具，集成网卡控制、802.1x 进程管理、移动热点(二维码分享)功能。

## 功能介绍

- **网卡管理**
  - 检测本地所有网络适配器的名称和启停状态。
  - 一键启用/禁用指定网卡。

- **8021x 进程控制**
  - 监控 8021x.exe 运行状态。
  - 结束进程并将可执行文件移动到指定目录；支持还原至原始路径。

- **热点管理**
  - 通过 Windows 原生接口（PowerShell 调用 WinRT API）开启/关闭移动热点。
  - 自定义热点名称（SSID）、密码、频段（2.4 GHz / 5 GHz），并保存配置。
  - 热点成功开启后自动生成 WiFi 二维码，方便手机扫码连接。

## 使用方法

### 1. 直接下载 Release 压缩包（推荐）

从 [Releases](https://github.com/wiizz3/ruijie_helper/releases/tag/V3)下载最新的 `校园网认证助手.zip` 文件，解压到任意目录，进入文件夹后**运行 `校园网认证助手.exe` **。

### 2. 克隆项目运行
- 克隆项目
```bash
git clone https://github.com/wiizz3/ruijie_helper.git
```
- 定位项目
```bash
cd ruijie_helper
```
- 安装依赖
```bash
pip install -r requirements.txt
```
- 运行程序
```bash
python main.py
```
